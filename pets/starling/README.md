# Starling

Starling is a star-eyed debugging companion with a moon tuft, blue-blonde anime-sticker energy, and a quietly focused build-and-review personality.

## Look

Starling is a compact 2D anime-sticker pet with huge starry eyes, a cream moon-shaped hair tuft, cyan-blue side hair, a black hoodie with a yellow star patch, and a small blue build-tool charm. The animation set includes idle, directional movement, waving, jumping, failed, waiting, active work, and review states.

## Files

- `pet.json`: Codex pet manifest.
- `spritesheet.webp`: 8x9 Codex pet atlas.
- `contact-sheet.png`: visual QA contact sheet.
- `previews/`: per-state animation previews.
- `validation.json` and `review.json`: deterministic QA output.

## Install

```bash
cp -R pets/starling ~/.codex/pets/starling
```

# Starry Avatar

Starry Avatar is a starry-eyed anime avatar pet based on a cartoon profile image, with a compact chibi silhouette and soft blue-blonde styling.

## Look

Starry Avatar is a compact anime-style pet with large starry eyes, blonde hair with bright blue accents, a dark oversized outfit, and expressive full-body poses. The animation set includes idle, directional movement, waving, jumping, failed, waiting, active work, and review states.

## Files

- `pet.json`: Codex pet manifest.
- `spritesheet.webp`: 8x9 Codex pet atlas.
- `contact-sheet.png`: visual QA contact sheet.
- `previews/`: per-state animation previews.
- `validation.json` and `review.json`: deterministic QA output.

## Install

```bash
cp -R pets/starry-avatar ~/.codex/pets/starry-avatar
```
